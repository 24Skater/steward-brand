# Screenshots

This directory contains screenshots for documentation, marketing, and app store listings.

## Naming Convention

```
{app}-{feature}-{variant}-{size}.png
```

Examples:
- `chms-dashboard-light-1280x720.png`
- `pos-checkout-dark-390x844.png`

## Required Sizes

### Web
- `1280x720` - Documentation/README
- `1920x1080` - Full HD marketing

### iOS App Store
- `1290x2796` - iPhone 15 Pro Max
- `1179x2556` - iPhone 15 Pro
- `2048x2732` - iPad Pro 12.9"

### Google Play
- `1080x1920` - Phone
- `1200x1920` - 7" Tablet
- `1600x2560` - 10" Tablet

## Guidelines

- Show realistic, useful content (not lorem ipsum)
- Capture both light and dark modes
- Include mobile and desktop views
- Keep sensitive data blurred or use demo data
