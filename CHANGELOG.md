# Changelog

All notable changes to the Steward Design System will be documented in this file.

This changelog is automatically updated by [Changesets](https://github.com/changesets/changesets).

## [Unreleased]

### @steward/tokens

- Initial release with DTCG-format design tokens
- Brand colors: Navy, Blue, Emerald, Amber, Red
- Semantic tokens for light and dark themes
- Typography scale (Display through Caption)
- Spacing scale (4px increments)
- Border radius tokens
- Shadow tokens
- CSS, TypeScript, and JSON output formats

### @steward/ui

- Initial release with React component library
- Core components:
  - Button (primary, secondary, destructive, outline, ghost, link)
  - Input, Textarea, Label, FormField
  - Select, Checkbox, Switch
  - Card (Header, Footer, Title, Description, Content)
  - Badge, Alert
  - Table
  - Dialog (Modal)
  - DropdownMenu
  - Tabs
  - Toast
  - Avatar, Skeleton, Tooltip, Progress
- Tailwind CSS preset with design tokens
- Full dark mode support
- Accessibility-first design

### @steward/email-templates

- Initial release with HTML email templates
- Receipt email template
- Welcome email template
- Shared email theme with brand tokens

### @steward/eslint-config

- Initial release with shared ESLint configuration
- Base config for all projects
- React-specific rules
- Node.js-specific rules

### @steward/tsconfig

- Initial release with shared TypeScript configurations
- Base config
- React app config
- Node.js config
- Library config
