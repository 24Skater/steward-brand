## Typography

**Inter** is our voiceâ€”clean, modern, and exceptionally readable. It scales beautifully from compact UI labels to bold headlines.

<br>

| Scale | Size | Weight | Use |
|-------|------|--------|-----|
| **Display** | 36px | Bold | Hero headlines |
| **H1** | 30px | Semibold | Page titles |
| **H2** | 24px | Semibold | Section headers |
| **H3** | 20px | Medium | Card titles |
| **Body** | 16px | Regular | Primary content |
| **Small** | 14px | Regular | Secondary text |
| **Caption** | 12px | Regular | Labels & metadata |

<br>

```
Font Stack: Inter, system-ui, -apple-system, sans-serif
```

### Logo Guidelines

- **Clear space**: Maintain padding equal to the height of the "S" on all sides
- **Minimum size**: Wordmark no smaller than 120px wide
- **Never**: Stretch, rotate, add shadows, apply gradients, or alter colors

## Get Started

<br>

### For Developers

```bash
# Install the design system
pnpm add @steward/tokens @steward/ui
```

```css
/* Import tokens in your app */
@import '@steward/tokens/dist/tokens.css';
```

```js
// Use the Tailwind preset
import stewardPreset from '@steward/ui/tailwind.preset';

export default {
  presets: [stewardPreset],
  // ...
};
```
